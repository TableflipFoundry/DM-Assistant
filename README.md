# D&D World Builder

A React-based desktop application that helps Dungeon Masters create rich content for their D&D universe using the OpenAI API.

## Features

- Generate detailed NPCs with D&D 5E stats and backgrounds
- Create immersive locations with points of interest
- Design factions with goals and relationships
- Generate historical events that shape your world

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file with:

```
VITE_OPENAI_API_KEY=your-api-key-here
```

## Tech Stack

- React with TypeScript
- Vite for building and development
- Material-UI for components
- OpenAI API for content generation
